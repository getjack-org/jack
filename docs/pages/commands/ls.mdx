# jack ls

List all your jack projects. This is an alias for `jack projects list`.

## Usage

```bash
jack ls
```

## Output

Shows projects grouped by directory:

```
  ~/dev/apps/
  ├── my-app      [live] https://alice-my-app.runjack.xyz
  └── test-api    [local only]

  ~/projects/
  └── wip-project [live] https://alice-wip-project.runjack.xyz

  On jack cloud (no local files)
  └── old-project [live] https://alice-old-project.runjack.xyz

ℹ 3 projects (2 live, 1 local-only)
```

## Status Indicators

| Status | Meaning |
|--------|---------|
| `[live]` | Deployed and accessible |
| `[local only]` | Exists locally but not deployed |
| `[error]` | Deployment failed |
| `[syncing]` | Currently deploying |
