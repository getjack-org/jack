# Commands

jack has a minimal command set by design. Here's the complete reference.

## Getting Started

| Command | Description |
|---------|-------------|
| [`new`](/commands/new) | Create a new project |
| [`ship`](/commands/ship) | Deploy your project |

## Projects

| Command | Description |
|---------|-------------|
| [`ls`](/commands/ls) | List all projects |
| [`projects`](/commands/projects) | Manage projects (info, remove, scan) |
| [`open`](/commands/open) | Open project in browser |

## Services & Secrets

| Command | Description |
|---------|-------------|
| [`services`](/commands/services) | Manage databases and storage |
| [`secrets`](/commands/secrets) | Manage project secrets |

## Support

| Command | Description |
|---------|-------------|
| [`feedback`](/commands/feedback) | Share feedback or report issues |

## Global Options

```bash
jack --help     # Show help
jack --version  # Show version
jack --debug    # Enable debug output
```
