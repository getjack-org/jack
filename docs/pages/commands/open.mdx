# jack open

Open your deployed project in the browser.

## Usage

```bash
jack open
```

Run from your project directory to open its live URL.

## What Happens

1. Looks up the project's deployed URL
2. Opens it in your default browser

## Examples

```bash
# Open current project
cd my-app
jack open

# Opens https://alice-my-app.runjack.xyz in your browser
```

## Notes

- Project must be deployed first (`jack ship`)
- Uses your system's default browser
