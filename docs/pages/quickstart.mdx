# Quickstart

Deploy your first project in under 2 minutes.

---

## 1. Create and Deploy

Pick a template and run a single command. No install needed.

### Option A: API

Build a REST API, webhook handler, or backend service.

```bash
bunx @getjack/jack new my-api -t api
```

**Includes**: Hono router, TypeScript, CORS, health endpoint

**Use for**: Webhooks, REST APIs, microservices

### Option B: AI Chat

Build a chatbot with streaming responses using free Cloudflare AI.

```bash
bunx @getjack/jack new my-chat -t ai-chat
```

**Includes**: Llama 3 (free), streaming UI, chat history

**Use for**: Chatbots, AI assistants, LLM experiments

### Option C: Farcaster Miniapp

Build a Farcaster Frame mini-app with React and database.

```bash
bunx @getjack/jack new my-miniapp -t miniapp
```

**Includes**: React + Vite, D1 database, Neynar API, Tailwind CSS

**Use for**: Farcaster frames, social apps, mini-games

**Requires**: [Neynar API key](https://neynar.com) (free tier available)

### Option D: Semantic Search

Build a RAG app with AI embeddings and vector search.

```bash
bunx @getjack/jack new my-search -t semantic-search
```

**Includes**: BGE embeddings (free), Vectorize database, D1 storage, search UI

**Use for**: Document search, knowledge bases, FAQ matching

### Option E: Interactive Selection

Not sure? Let jack help you choose:

```bash
bunx @getjack/jack new my-app
```

Jack will prompt you to select a template interactively.

Jack handles everything: scaffolding, dependencies, and first deploy. Your app is live before you write a line of code.

---

## 2. Make Changes and Redeploy

Edit your code, then ship again:

```bash
cd my-app

jack ship
```

Each deploy takes seconds. Your changes are live immediately.

---

## What's Next?

### Add Services

Need a database, storage, or AI?

```bash
jack services db

jack services kv

jack services r2
```

### Set Secrets

Store API keys securely:

```bash
jack secrets
```

### Open Your Project

Quick access to your deployed app:

```bash
jack open

jack open --editor
```

### Explore Commands

See all available commands:

```bash
jack --help
```

## Fork Any Published Project

See a project you like? Fork it:

```bash
jack new my-fork -t username/project-name
```

This creates your own copy that you fully control.

---

## Alternative: Global Install

If you prefer having jack available globally:

```bash
bun add -g @getjack/jack
```

Then use `jack new my-app` directly without `bunx`.
