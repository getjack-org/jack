Your AI wrote the app. Cursor built the API. Claude scaffolded the whole thing.

Now it's rotting in localhost because deploys are *a whole thing*.

**`jack new my-app`** → live URL in 30 seconds.

No config. No accounts. No friction.

```bash
bun add -g @getjack/jack
```

Then: `jack new my-app`

Or try without installing: `bunx @getjack/jack new my-app`

## The problem

You built 47 things this month.

3 made it to production.

The rest died in localhost because "I'll deploy it later."

Later never comes.

## The fix

`jack new` creates AND deploys. One command. Live URL. Done.

`jack ship` redeploys. No ceremony. No config files. No friction.

Your experiments stop dying. Your portfolio grows. You ship.
