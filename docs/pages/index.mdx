## **Ship before you forget why you started.**

jack is a CLI for vibecoders who want to deploy to the cloud without thinking about infrastructure. One command to create, one command to ship.

<img src="/jack-demo.gif" alt="jack demo" style={{ margin: '2rem 0', borderRadius: '12px', width: '100%' }} />

## Quick Start

```bash
# Install
bun add -g @getjack/jack

# Create a new project
jack new my-app

# Deploy it
jack ship
```

## Why jack?

- **Zero config** - No YAML files, no dashboards, no decisions
- **AI-native** - Built-in MCP server for Claude Code and other agents
- **Fast** - Deploys to edge in seconds
- **Simple** - Three commands: `new`, `ship`, `open`

## Commands

| Command | What it does |
|---------|--------------|
| `jack new` | Create a new project from a template |
| `jack ship` | Deploy your project |
| `jack ls` | List your projects |
| `jack open` | Open your project in the browser |
